## Start

Go to [playit.gg](https://playit.gg) and log in with any account.

## Set up playit.gg

1. Download the agent from [playit.gg/download](https://playit.gg/download).
2. Run the agent on the host PC.
3. Create a custom **TCP** tunnel with **port count 3**.
4. Set local IP to `127.0.0.1` and local port to the Blender host port (`5555` by default).
5. Keep the agent running.

## Tunnel settings

| Field | Value |
| --- | --- |
| Protocol / tunnel type | **TCP** |
| Port count | **3** |
| Local IP | `127.0.0.1` |
| Local port | Blender **Host** port (`5555` by default) |
| Proxy Protocol | **None** or **Disabled** |
