This site documents **[Multi-user Playit](https://github.com/sin-boo/multi-user-playit)** — a [playit.gg](https://playit.gg)-optimized fork of [Multi-User for Blender](https://gitlab.com/slumber/multi-user) by Swann Martinez.

Use this fork for internet hosting over a playit.gg tunnel. For general addon usage on a local network, see the [original Multi-User docs](https://slumber.gitlab.io/multi-user/index.html).

## How it works

The fork keeps the same collaborative Blender workflow but replaces manual port forwarding with a playit.gg **TCP** tunnel:

1. Install the Multi-user Playit addon in Blender.
2. Set up playit.gg on the host PC — account, Premium subscription, agent, and tunnel.
3. Host a session in Blender and share the public playit address with joiners.

## Install the addon

No official release yet — install from the GitHub repository:

1. Clone or download [sin-boo/multi-user-playit](https://github.com/sin-boo/multi-user-playit).
2. Zip the `multi_user` folder, or use a pre-built `multi_user-*.zip` from [Releases](https://github.com/sin-boo/multi-user-playit/releases) if one is available.
3. In Blender: **Edit → Preferences → Add-ons → Install from disk** → select the zip.

Dependencies are installed into Blender's Python automatically during setup.

After playit.gg is configured:

1. In Blender, open the **Multi-User** panel.
2. Under **Hosting**, set your port (default `5555`) and click **Host**.
3. Share the playit join address (e.g. `my-tunnel.gl.at.ply.gg:41234`) with joiners.

### Join from another PC

Joiners also need the addon installed. Then:

1. In Blender, open **Multi-User → Server list**.
2. Click **+ Add server preset**.
3. Paste the **full** playit address into the **IP** field (e.g. `my-tunnel.gl.at.ply.gg:41234`). The port is parsed automatically.
4. Click **Connect**.
