**Multi-user Playit** is a [playit.gg](https://playit.gg)-optimized fork of [Multi-User for Blender](https://gitlab.com/slumber/multi-user). It lets several people work on the same Blender scene over the network without port forwarding or manual firewall setup.

Install the addon, run the playit.gg agent on the host machine, create a TCP tunnel with the settings in this guide, and share the public address so others can join.

This project is under active development. It has been tested on Windows; use at your own risk.
</Warning>

<Note>
TCP tunnels require Premium on current playit.gg plans. Download and sign in to the agent before creating your tunnel.
</Note>

## Quick start

1. [Download](https://playit.gg/download) and sign in to the playit.gg agent.
2. [Subscribe to Premium](https://playit.gg/account/billing/shop/premium) if you do not already have it.
3. [Create the TCP tunnel](/playit-tunnel) with port count **3** and local port **5555** (or your Blender host port).
4. In Blender: Multi-User panel → set **Port** → **Host**.
5. Share the playit address (e.g. `my-tunnel.gl.at.ply.gg:41234`) with joiners.
