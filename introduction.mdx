Welcome. This repository is the source for a [Mintlify](https://mintlify.com) documentation site.

Edit pages as `.mdx` files in this repo, update navigation in `docs.json`, and push to `main` to publish.

## What you can document here

Use this site for product guides, architecture notes, runbooks, and internal references. Add new pages under the repo root (or a folder) and list them in `docs.json`.
