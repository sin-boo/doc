## Overview

**Multi-user Playit** is a version of the original [Multi-User](https://gitlab.com/slumber/multi-user) Blender addon, maintained at [github.com/sin-boo/multi-user-playit](https://github.com/sin-boo/multi-user-playit). The original lets several people work on the same `.blend` file over the network. Multi-user Playit keeps that same idea but uses [playit.gg](https://playit.gg) so hosting and joining a session over the internet is simpler — no port forwarding or manual firewall setup on the host.

**For you (host):** run the playit.gg agent, create a TCP tunnel, and click **Host** in Blender.

**For others (joiners):** install the addon, paste the tunnel address the host gives you, and connect.

These docs cover that setup. The [original Multi-User docs](https://slumber.gitlab.io/multi-user/index.html) still apply for the Blender panel, LAN sessions, and general addon behavior.

## Requirements

**Host PC**

- Blender with Multi-user Playit installed
- playit.gg account, agent installed and signed in
- playit.gg subscription (required for custom TCP tunnels on current plans)
- Tunnel configured per [Create a playit.gg tunnel](/playit-tunnel)

**Joiner PC**

- Blender with Multi-user Playit installed
- Public playit address from the host (hostname + port)

## Install the addon

There is no official release package yet. Install manually:

1. Download the repo: [sin-boo/multi-user-playit](https://github.com/sin-boo/multi-user-playit) (clone or zip).
2. Create a zip of the `multi_user` folder inside the repo, or download `multi_user-*.zip` from [Releases](https://github.com/sin-boo/multi-user-playit/releases) if listed.
3. In Blender: **Edit → Preferences → Add-ons → Install from disk** → select the zip → enable the addon.

The installer adds required Python dependencies to Blender automatically.

Repeat on every machine that will host or join.

## Set up a session

Work through these steps in order.

### Host

1. Install the addon (above).
2. Install the [playit.gg agent](https://playit.gg/download) on the host PC and sign in.
3. Ensure an active [subscription](https://playit.gg/account/billing/shop/premium) on the host account.
4. Create a TCP tunnel with **port count 3** and local port **5555** (or your chosen host port). Full field values are in [Create a playit.gg tunnel](/playit-tunnel).
5. Leave the agent running.
6. In Blender, open the **Multi-User** panel → **Hosting** → set **Port** (must match tunnel local port) → **Host**.
7. Copy the tunnel's public address from playit (example: `my-tunnel.gl.at.ply.gg:41234`) and send it to joiners.

### Join

1. Install the addon (above).
2. Open **Multi-User** → **Server list** → **+ Add server preset**.
3. Paste the host's full playit address into **IP** (port is parsed from the string).
4. Click **Connect**.

If hosting or joining fails, see [Troubleshooting](/playit-tunnel#troubleshooting) on the tunnel page.
