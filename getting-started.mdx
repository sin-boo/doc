[Multi-user Playit](https://github.com/sin-boo/multi-user-playit) is a [playit.gg](https://playit.gg) fork of [Multi-User for Blender](https://gitlab.com/slumber/multi-user). Install the addon in Blender, then follow these steps on the host PC.

<Note>
A playit.gg **Premium** subscription is required for TCP tunnels.
</Note>

## 1. Create an account

Go to [playit.gg](https://playit.gg) and click **Get started**.

<img src="/images/playit-homepage.png" alt="playit.gg homepage" />

## 2. Subscribe to Premium

Go to [playit.gg/pricing](https://playit.gg/pricing) and choose **Premium**.

<img src="/images/playit-pricing.png" alt="playit.gg pricing page" />

## 3. Download and run the agent

Go to [playit.gg/account/agents](https://playit.gg/account/agents) and download the **Windows Installer (64-bit)**.

<img src="/images/playit-download-agent.png" alt="playit.gg agent download page" />

Run the installer, then open **Playit.gg** from the Windows Start menu.

<img src="/images/playit-windows-search.png" alt="Windows search for Playit.gg" />

Sign in and complete setup until your agent is connected. Click **Create a tunnel**.

<img src="/images/playit-agent-connected.png" alt="playit.gg agent connected" />

## 4. Create a tunnel

Create a custom **TCP** tunnel with these settings:

| Setting | Value |
| --- | --- |
| Protocol | TCP |
| Port count | 3 |
| Local IP | `127.0.0.1` |
| Local port | `5555` |
| Proxy protocol | None |

When playit asks setup questions, use answers like **multiplayer blender** for the software description, confirm lawful use, and pick the **Premium Network** datacenter closest to you.

Copy your public join address (for example, `my-tunnel.gl.at.ply.gg:41234`) and share it with joiners.

## 5. Host in Blender

1. Open the **Multi-User** panel.
2. Set port to `5555` and click **Host**.
3. Share the playit address with anyone joining.

To join from another PC: open **Server list**, add a server preset, paste the full playit address, and click **Connect**.
