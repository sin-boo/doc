[Multi-user Playit](https://github.com/sin-boo/multi-user-playit) is a [playit.gg](https://playit.gg) fork of [Multi-User for Blender](https://gitlab.com/slumber/multi-user). Install the addon in Blender, then follow these steps on the host PC.

<Note>
A playit.gg **Premium** subscription is required for TCP tunnels.
</Note>

## 1. Create an account

<Columns cols={2}>
  <div>

<Columns cols={2}>

## 3. Download and install the agent

<Columns cols={2}>
  <div>
    Go to [playit.gg/account/agents](https://playit.gg/account/agents). Download the installer for your system — **Windows Installer (64-bit)** is recommended for Windows 10/11. Run the installer to complete setup.

<Columns cols={2}>
  <div>

## 6. Create a tunnel

<Columns cols={2}>
  <div>
    Fill in the tunnel form with these settings:

    - **Name:** `blender` (or any name you prefer)
    - **Tunnel type:** TCP
    - **Port count:** 3
    - **Software description:** `blender for addon`
    - **Usage confirmation:** Confirmed
    - **Public endpoint:** Premium Network — closest datacenter to you
    - **Assign to agent:** Your connected agent
    - **Local IP:** `127.0.0.1`
    - **Local port:** `5555`
    - **Proxy protocol:** None

    Click **Next** to finish.
  </div>
  <img
    src="/images/playit-tunnel-setup.png"
    alt="playit.gg tunnel setup form"
    noZoom
    className="rounded-lg"
    style={{ width: "100%", maxWidth: "360px", height: "auto" }}
  />
</Columns>

## 7. Copy your join address

<Columns cols={2}>
  <div>
    After the tunnel is created, open the **Tunnels** page. Copy the address shown in the **orange box** at the top (for example, `protocols-yards.with.playit.plus:1209`). Share this with anyone joining your session.
  </div>
  <img
    src="/images/playit-tunnel-address.png"
    alt="playit.gg tunnel page showing the public address in the orange box"
    noZoom
    className="rounded-lg"
    style={{ width: "100%", maxWidth: "360px", height: "auto" }}

## 8. Host in Blender

<Columns cols={2}>
  <div>
    Open **Playit.gg**, then in Blender open the **Multi-User** panel. Make sure port is set to `5555` and click **Host**.

    After the first setup, this is all you need to do each time you want to host — open Playit.gg and press **Host**. You do not need to repeat the steps above.
  </div>
  <img
    src="/images/blender-host.png"
    alt="Blender Multi-User panel with the Host button"
    noZoom
    className="rounded-lg"
    style={{ width: "100%", maxWidth: "360px", height: "auto" }}
  />
</Columns>

## 9. Join a session

<Columns cols={2}>
  <div>
    On the joining PC, open the **Multi-User** panel in Blender:

    1. Click the **+** button to add a server preset.
    2. Paste the playit address into the **IP** field (the address from the orange box, e.g. `protocols-yards.with.playit.plus:1209`).
    3. Click **OK**, then click **Connect**.

    Once the server is saved, you can reconnect anytime without re-entering the address.
  </div>
